# 云改商机上报支撑系统API接口文档

------

## 1.登录接口

- 请求路径：/doLogin
- 请求方法：post
- 请求参数

| 参数名    | 参数说明 | 备注     |
| --------- | -------- | -------- |
| accountId | 用户名   | 不能为空 |
| pass      | 密码     | 不能为空 |

- 响应参数

| 参数名 | 参数说明     | 备注 |
| ------ | ------------ | ---- |
| msg    |              |      |
| status | 登录验证结果 |      |

- 响应数据

  ```javascript
  {"msg":"","status":"ok"}
  ```
